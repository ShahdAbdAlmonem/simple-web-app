## Setup
1. Install dependencies: `npm install`
2. Run the app: `npm start`
3. Access at `http://localhost:3000`

## Jenkins Pipeline
- Checkout code from Git.
- Install dependencies.
- Build and test.
- Deploy locally.

## Requirements
- Node.js
- Jenkins
- Git
